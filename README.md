Sistem Kasir



Deskripsi



Sistem Kasir adalah aplikasi sederhana berbasis HTML, CSS, dan JavaScript yang saya buat sebagai proyek UAS mata kuliah Git dan GitHub. Aplikasi ini digunakan untuk membantu proses pengelolaan data barang, pencarian barang, transaksi, dan melihat laporan penjualan.



Fitur



\- Login

\- Dashboard

\- Data Barang (CRUD)

\- Pencarian Barang

\- Transaksi

\- Laporan

\- Logout
